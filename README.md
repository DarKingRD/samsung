# Система исправления опечаток

Итоговый проект курса Samsung по ИИ.

## Описание

Система для исправления опечаток разной природы:
- Орфографические ошибки
- Грамматические ошибки (запятые, согласование слов)
- Смысловые ошибки
- Стилистические ошибки

## Структура проекта

```
.
├── data/              # Датасеты
├── models/            # Сохранённые модели
├── src/

│   ├── model.py              # Архитектура модели
```

## Установка

### Быстрая установка:
```bash
python setup.py
```

### Ручная установка:
```bash
pip install -r requirements.txt
python -m nltk.downloader punkt
```

## Использование

### Шаг 1: Подготовка данных
Скачивает и обрабатывает датасеты:
```bash
python src/data_processing.py
```

### Шаг 2: Обучение модели
Обучает модель на обработанных данных:
```bash
python src/train.py
```

### Шаг 3: Запуск веб-интерфейса
Запускает веб-сервер:
```bash
python web/app.py
```

Откройте браузер по адресу **http://localhost:5000**

## Структура проекта

- `src/data_processing.py` - обработка датасетов
- `src/model.py` - архитектура модели
- `src/train.py` - обучение модели
- `src/inference.py` - инференс и исправление текста
- `web/app.py` - Flask веб-сервер
- `web/templates/index.html` - веб-интерфейс
- `web/static/style.css` - стили

## Датасеты

Проект использует следующие датасеты:
1. GitHub Typo Corpus - многоязычный датасет с опечатками
2. Kartaslov - русскоязычный датасет орфографических ошибок
